## About The Project

This project is aimed for readers who are in need of tracking what books they read and waht books they haven't read.

Users should be able to:

- Create account using username and password
- Add new Books
- Update Books
- Get Books
- Delete Books

## Built with

- Node JS
- Express
- Swagger UI
